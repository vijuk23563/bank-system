# BANK SYSTEM /
This is an simple CLI based bank system.
There are some functions -
         1. CREATE ACCOUNT
         2. DEPOSIT MONEY 
         3. WITHDRAW MONEY
         4. BALANCE ENQUIRY
         5. ACCOUNTS LIST
         6. DELETE ACCOUNT
         7. MODIFY ACCOUNT
         AND EXIT TO CLOSE THIS SYSTEM 
This project is for education purpose and this is simple level project..
